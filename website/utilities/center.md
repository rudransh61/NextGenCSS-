# Center

## Syntax (custom)

````sass
```other code
    center : "center",
```
````

## Syntax

`.center`

Puts element in center horizontally

## Compiled as

```scss
.center {
    display: block;
    margin-left: auto;
    margin-right: auto;
    width: 40%;
}
```
